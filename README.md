# TRPG 成就卡片锻造炉

一款免费、纯前端的 TRPG 成就卡片生成工具，帮助游戏主持人和玩家快速生成精美的横幅式成就卡片。

## ✨ 功能特性

- 🎨 **实时预览** - 填写即预览，所见即所得
- ⚡ **动态金字** - 波光粼粼的鎏金闪烁成就名称
- 🖼️ **自定义图标** - 支持上传个人图标，自动裁剪为圆形
- 🎭 **多种背景主题** - 暗黑、羊皮纸、赛博朋克、星空、极简
- 📤 **导出 PNG** - 高清 3x 像素比导出
- 💾 **导出/导入 JSON** - 工程文件支持，随时恢复编辑
- 📚 **历史记录** - 自动保存到 IndexedDB，最多 50 条记录

## 🚀 本地运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 🛠️ 技术栈

- **框架**: Vue 3 + Vite
- **样式**: Tailwind CSS
- **状态管理**: Pinia
- **图片导出**: html-to-image
- **本地存储**: localForage (IndexedDB)
- **字体**: 霞鹜文楷 (LXGW WenKai)
