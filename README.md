# ⚔️ 成都秘密基地 TRPG 成就卡生成器

一款为跑团量身定制的精美成就卡锻造工具，旨在记录每一段难忘的冒险旅程。

---

## ✨ 核心亮点

- 🛡️ **双向版式**：支持传统的**横版**与精致的**竖版**卡组样式。
- 🏆 **动态金字**：自动生成波光粼粼的鎏金动态闪烁成就标题。
- 🎨 **深度自定义**：
    - 多种预设主题（暗黑、羊皮纸、赛博、星空、极简）。
    - 支持上传**背景图片**与**成就图标**（自动裁圆）。
    - 调色盘自定义边框颜色，支持 0-10px 边框粗细微调。
- 📤 **数据嵌入式导出**：
    - 支持导出 **PNG/GIF**，图片中自动嵌入成就 JSON 元数据。
    - 上传带数据的卡片即可瞬间还原编辑现场。
- 💾 **本地成就馆**：内置 IndexedDB 自动保存 50 条历史记录，配备排序与**记录河流（时间线）**展示。

## 🛠️ 驱动核心

- **Vue 3 + Vite**：极速响应的前端架构。
- **Pinia**：灵活的状态管理。
- **html-to-image & gifshot**：高清卡片与动态记录生成。
- **LocalForage**：可靠的本地数据持久化。
